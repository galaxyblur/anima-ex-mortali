# Level 5: User Stories Overview

**Document Type:** Validation & Tuning
**References:** All levels (1-4) + External

---

## Organization

Stories organized by priority:

| Priority | Description | Status |
|----------|-------------|--------|
| **MVP** | Essential for v1 launch | Active |
| **SECONDARY** | Future enhancements | Deferred |

## Story Index

### MVP

**Creator Stories:**
- US-001: Add charter page
- US-002: Add code/app artifact with external link
- US-003: Add prose artifact with embedded content
- US-004: Add poetry artifact
- US-005: Add music artifact with streaming link
- US-006: Link artifact to thread
- US-007: Create new thread
- US-008: Write changelog entry

**Visitor Stories:**
- US-010: View charter
- US-011: Browse all artifacts
- US-012: Filter artifacts by medium
- US-013: View single artifact with thread context
- US-014: Browse thread and see linked artifacts
- US-015: View changelog (what's new)

### Secondary

- US-020: Search artifacts
- US-021: Contact form
- US-022: Email newsletter subscription

## Test Approach

Given the static site nature, stories focus on:
- Content renders correctly
- Navigation works between pages
- Filtering produces correct results
- Links function

No complex state management or user authentication to test.
